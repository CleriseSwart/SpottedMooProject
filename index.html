What is Machine Embroidery Digitizing?

Embroidery digitizing is the process of converting artwork into a digital file using a software that allows embroidery machines to understand the needle's path.

Is this an automatic process?

This process is not automated and in fact, great digitizing is considered an artform if done correctly.

What is Logo Digitization?

Read more here

Embroidery machines are not able to read the same types of files as our computer can, so you need to in essence redraw the logo/image in a way that the machine can understand before they can do their job. From a digitizers perspective you take a logo in either PNG or JPG format and redraw/recreate the image in an embroidery file.

What formats do Embroidery Machines use?

Every machine embroidery file format was created by different embroidery machine companies which is why there are so many different ones. And, in some cases the file format for a specific brand of machine has evolved, like the VIP to VP3

  • .dst: embroidery file format for Tajima commercial embroidery machines
  • .exp: embroidery file format for Melco commercial embroidery machines
  • .jef: embroidery file format for Janome commercial embroidery machines
  • .kwk: embroidery file format for Brother commercial embroidery machines
  • .dsb: embroidery file format for Barudan commercial embroidery machines
  • .tap: embroidery file format for Happy commercial embroidery machines

What types of stitches are there?

There are 3 basic types of stitches

Straight Stitches

These are the foundation of Embroidery. Straight lines of stitches are repeated to make a pattern. You can create thick lines by repeating a straight stitch many times. It is important to note that straight stitches are not just for straight lines, they can be used for curved shapes too!

Straight Stitches are most commonly used for:
  • Shading
  • Outlining
  • Detail Work

Satin Stitches

If you are looking to have text embroidered, then the satin stitch is probably the stitch type for you. The longer thread length of satin stitching gives it a shiny look that works perfectly for lettering. The maximum stitch length for satin stitches is 12.1mm.

Satin Stitches are most commonly used for:
  • Words and text
  • Borders
  • Achieving a shiny finished look

Fill Stitches

Also known as the tatami stitch or the ceding stitch, the fill stitch is typically used to fill in patterns, just like its name suggests. Fill stitches use alternating lines to give the embroidery a woven appearance. The maximum stitch length for fill stitches is 4mm.

Fill Stitches are most commonly used for:
  • Filling in larger blank space
  • Creating a textured look
  • Larger designs
  • More durable applications
